# ライントレース・障害物回避ロボット

`arduino/main.cpp` は、黒地にある白線を追従し、前方の障害物を超音波センサーで検出して回避するロボット用プログラムです。

## 使用する入出力

| 用途 | ピン・機器 |
| --- | --- |
| ラインセンサー | S0〜S4（A0〜A4）。S0は右端、S4は左端 |
| 左モーター速度 | D11（PWM） |
| 右モーター速度 | D9（PWM） |
| モーター方向 | D12、PA_11_ALT0 |
| 超音波センサーTRIG | D7 |
| 超音波センサーECHO | D8 |

## 白線追従アルゴリズム

実機のコースは黒地に白線であるため、各反射センサーの値が `WHITE_THRESHOLD`（初期値0.65）以上であれば白線を検出したと判断します。

各センサーに、右から順に `+2, +1, 0, -1, -2` の位置を割り当てます。白線を検出したセンサー位置の平均を「ずれ量」として求め、左右モーターの速度差を作ります。

```text
左モーター速度 = BASE_SPEED + TURN_GAIN × ずれ量
右モーター速度 = BASE_SPEED - TURN_GAIN × ずれ量
```

- 白線が中央（S2）にある場合は、左右同じ基本速度で直進します。
- 白線が右側（S0、S1）にある場合は、左モーターを速くして右へ曲がります。
- 白線が左側（S3、S4）にある場合は、右モーターを速くして左へ曲がります。
- 全センサーで白線を見失った場合は、最後に検出した方向へ低速で旋回して白線を探します。

このため、カーブでも白線の位置に応じて連続的に旋回量が変化します。急カーブで曲がり切れない場合は、`BASE_SPEED` を下げるか、`TURN_GAIN` を上げて調整します。

## 超音波測距

超音波センサーはD7から10マイクロ秒のトリガーパルスを出力し、D8のエコーパルス幅を測定します。音速を約0.0343 cm/マイクロ秒として、往復時間の半分から前方の距離を計算します。

エコーを30 ms以内に受信できない場合は、測距失敗として `-1` を返します。これにより、エコー未受信でプログラムが停止し続けることを防ぎます。測距は80 msごとに行い、最新の距離をデバッグコンソールへ1秒ごとに表示します。

## 障害物回避アルゴリズム

前方の距離が `OBSTACLE_DISTANCE_CM`（初期値18 cm）未満なら障害物と判断し、以下の状態遷移で回避します。

1. **停止**：200 ms停止します。
2. **後退**：150 ms後退して、旋回のための空間を作ります。
3. **右旋回**：380 ms右へ旋回して、障害物の横へ移動します。
4. **通過**：750 ms直進して障害物の横を通過します。
5. **左旋回**：380 ms左へ旋回して、元のライン方向へ戻ります。
6. **白線再探索**：1.2秒間、通常の白線追従でラインへの復帰を試みます。
7. **通常走行**：白線追従と障害物監視を再開します。

回避を開始すると、2.5秒間は障害物の再検出を無効にします。これにより、回避中に同じ障害物を再検出して、回避処理を繰り返すことを防ぎます。

## 実機で調整する値

`arduino/main.cpp` 冒頭の定数を、実機の速度・センサー値・障害物の大きさに合わせて調整します。

| 定数 | 内容 |
| --- | --- |
| `WHITE_THRESHOLD` | 白線と判定する反射センサー値 |
| `BASE_SPEED` | 白線追従時の基本速度 |
| `TURN_GAIN` | 白線のずれに対する旋回量 |
| `SEARCH_SPEED` | 白線を見失ったときの探索速度 |
| `OBSTACLE_DISTANCE_CM` | 障害物と判断する距離 |
| `ULTRASONIC_TIMEOUT` | エコー受信を待つ最長時間 |
| `DISTANCE_SAMPLE_INTERVAL` | 超音波測距の間隔 |
| `OBSTACLE_REARM_DELAY` | 回避後に障害物検出を再開するまでの時間 |

障害物回避の各時間（停止・後退・旋回・通過）は、`runAvoidance()` 内で設定しています。障害物の位置、ロボットの速度、路面の摩擦に応じて実機で調整してください。

## 注意

HC-SR04など、ECHO端子が5 V出力の超音波センサーを使う場合は、マイコンの3.3 V入力へ直接接続せず、分圧回路などでレベル変換してください。
